# 路盟 LUMOGUIDE — 页面目录

> 共 69 个路由页面，覆盖 17 个功能模块。基于 Vue 3 CDN + Hash 路由。

---

## Tab 页（5 个）

| 页面名字 | 页面功能 |
|---------|---------|
| 首頁（Home） | 搜索入口、城市攻略（6 列网格）、大洲分组城市（自动轮播）、导游分类（5s 自动切换）、商家区块（Banner 轮播驱动分类切换）、资讯列表 |
| 城市（City） | 服务器驱动的大洲+地区二级筛选、4 列城市卡片网格、导游发布城市入口按钮 |
| 資訊（News） | 资讯文章列表、分类筛选（封面卡片布局、衬线标题、加载更多） |
| 消息（Message） | 消息中心：5 类消息入口卡片（系统/关注/评论/预约/互动），红色未读角标 |
| 我的（Mine） | 个人资料卡片、服务菜单网格、VIP 会员卡、推广卡片、退出登录 |

## 认证页（6 个）

| 页面名字 | 页面功能 |
|---------|---------|
| 歡迎（Welcome） | 品牌开场动画：手写逐字出现（clip-path 动画），中英文自动切换，停留 2s 后自动跳转（已登录→首页，未登录→登录） |
| 登錄（Login） | 邮箱+密码登录、记住密码（localStorage 持久化）、App 图标展示、SVG 简笔画图标、表单触发浏览器密码管理器 |
| 註冊（Register） | 邮箱+验证码+密码注册、用户协议勾选、inline 发送验证码按钮 |
| 忘記密碼（ForgetPassword） | 邮箱验证 → 重置密码 |
| 驗證碼（VerifyCode） | 验证码输入（注册/找回密码流程共用） |
| 設置密碼（PasswordInput） | 新密码设置（找回密码流程最后一步） |
